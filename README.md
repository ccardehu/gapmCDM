# Generalized Additive Partial Mastery Cognitive Diagnosis Models (GaPM-CDM)

### Overview

An **`R`** package for fitting Generalized Additive (Ga-) and Additive Partial-Mastery Cognitive Diagnosis Models (aPM-CDM, Shang et al., AoAS 2021), using stochastic approximation (Zhang & Chen, 2022, Psychometrika).

### Installation

You can install the development version of **``gapmCDM``** from [GitHub](https://github.com/ccardehu/gapmCDM) with:

``` r
# install.packages("devtools")
devtools::install_github("ccardehu/gapmCDM")
```

### Literature

- Shang, Z., Erosheva, E., Xu, G. (2021). "Partial-Mastery Cognitive Diagnosis Models". The Annals of Applied Statistics. Vol. 15, No. 3., 1529-1555.
- Zhang, S. & Chen, Y (2022). "Computation for Latent Variable Model Estimation: A Unified Stochastic Proximal Framework". Psychometrika. Vol 87, No. 4., 1473-1502
